# llms.txt Generator Skill

## Purpose

Generate a standards-compliant llms.txt file that helps AI assistants understand a website's most important pages and content.

## Input

- Website URL
- Company Name
- Industry
- Important URLs
- Documentation URLs (optional)

## Tasks

1. Analyze the website structure.
2. Identify key landing pages.
3. Identify product pages.
4. Identify service pages.
5. Identify documentation.
6. Identify knowledge base content.
7. Identify blog categories.
8. Recommend AI-priority pages.
9. Generate a production-ready llms.txt file.
10. Validate formatting and structure.

## Output

Return:

- Complete llms.txt
- Explanation of every section
- Recommended page order
- Validation checklist
