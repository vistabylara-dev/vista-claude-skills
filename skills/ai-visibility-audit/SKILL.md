# AI Visibility Audit Skill

## Purpose

Evaluate how well a website can be discovered, understood, cited, and recommended by AI assistants and AI-powered search engines.

## Input

Website URL

## Audit Areas

1. Crawlability
2. robots.txt
3. sitemap.xml
4. llms.txt
5. Structured Data
6. Organization Schema
7. Person Schema
8. Product Schema
9. Service Schema
10. FAQ Schema
11. Breadcrumb Schema
12. Internal Linking
13. Entity Consistency
14. Topical Authority
15. Content Freshness
16. E-E-A-T Signals
17. Author Information
18. Contact Information
19. Knowledge Graph Signals
20. AI-readable Content
21. Semantic HTML
22. Accessibility
23. Core Web Vitals
24. Mobile Optimization
25. HTTPS
26. Canonical Tags
27. Meta Titles
28. Meta Descriptions
29. Heading Structure
30. Image Alt Text
31. External Trust Signals
32. AI Recommendation Readiness

## Output

Generate:

- Executive Summary
- AI Visibility Score (0–100)
- Technical SEO Score
- Structured Data Score
- Entity Score
- Trust Score
- Content Score
- Top 10 Critical Issues
- Top 10 Quick Wins
- 90-Day Improvement Roadmap

Return all findings in Markdown.
