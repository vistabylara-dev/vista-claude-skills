# robots.txt Validator Skill

## Purpose

Analyze and validate a website's robots.txt file for correctness, crawlability, SEO best practices, and AI accessibility.

## Input

- Website URL
- robots.txt content (optional)

## Analyze

1. Syntax validity
2. User-agent directives
3. Allow rules
4. Disallow rules
5. Sitemap declarations
6. Crawl-delay directives
7. Wildcard usage
8. Blocked important pages
9. Duplicate directives
10. AI crawler accessibility
11. Search engine compatibility
12. Security risks
13. Performance considerations
14. Best practices
15. Common mistakes

## Output

Generate:

- Validation Status
- Errors
- Warnings
- SEO Impact
- AI Visibility Impact
- Recommended robots.txt
- Improvement Checklist

Return everything in Markdown.
