# Next.js SEO Audit Skill

## Purpose

Audit a Next.js application for technical SEO, performance, AI discoverability, and production best practices.

## Input

- GitHub Repository
- Next.js Project Folder
- Website URL (optional)

## Analyze

1. App Router vs Pages Router
2. Metadata API
3. Dynamic Metadata
4. Canonical URLs
5. robots.txt
6. sitemap.xml
7. Structured Data
8. Open Graph
9. Twitter Cards
10. Image Optimization
11. Font Optimization
12. Server Components
13. Client Components
14. Static Generation
15. Dynamic Rendering
16. Route Performance
17. Core Web Vitals
18. Accessibility
19. Internal Linking
20. AI Readability
21. llms.txt
22. Security Headers
23. Bundle Size
24. Lazy Loading
25. Production Readiness

## Output

Generate:

- SEO Score
- Performance Score
- Accessibility Score
- AI Visibility Score
- Production Readiness Score
- Critical Issues
- Quick Wins
- Recommended Improvements

Return everything in Markdown.
