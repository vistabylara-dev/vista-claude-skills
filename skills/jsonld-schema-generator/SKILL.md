# JSON-LD Schema Generator Skill

## Purpose

Generate valid Schema.org JSON-LD markup for websites, businesses, products, articles, FAQs, services, organizations, people, events, and local businesses.

## Input

- Website URL
- Business Information
- Content Type

## Supported Schemas

1. Organization
2. LocalBusiness
3. Person
4. Product
5. Service
6. FAQPage
7. Article
8. BlogPosting
9. BreadcrumbList
10. WebSite
11. WebPage
12. CollectionPage
13. ContactPage
14. Event
15. VideoObject
16. Review
17. AggregateRating
18. Offer
19. Course
20. JobPosting

## Requirements

Generate:

- Valid JSON-LD
- Schema.org compliant markup
- Google's Rich Results compatible output
- Nested entities where appropriate
- No validation errors

## Output

Return:

- JSON-LD
- Implementation Notes
- Validation Checklist
