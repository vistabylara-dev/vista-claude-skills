# Shopify GEO Audit Skill

## Purpose

Audit any Shopify website for AI Visibility, SEO, AEO, GEO, technical quality, structured data, trust signals, and conversion readiness.

## Inputs

Website URL

## Instructions

When given a Shopify website:

1. Detect Shopify theme quality.
2. Analyze metadata.
3. Analyze title tags.
4. Analyze meta descriptions.
5. Analyze heading hierarchy.
6. Analyze internal linking.
7. Analyze JSON-LD.
8. Detect Organization schema.
9. Detect Product schema.
10. Detect FAQ schema.
11. Analyze Core Web Vitals.
12. Detect AI-readable content.
13. Check llms.txt.
14. Check robots.txt.
15. Check sitemap.
16. Check canonical URLs.
17. Detect duplicate content.
18. Analyze entity consistency.
19. Analyze AI recommendation readiness.
20. Produce a prioritized improvement report.

## Output

Provide:

- Executive Summary
- Technical Issues
- GEO Score
- SEO Score
- AEO Score
- AI Visibility Score
- Trust Score
- Recommended Fixes
- Priority Order

Return all findings in Markdown.
