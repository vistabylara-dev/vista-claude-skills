# Shopify GEO Audit

A production-ready Claude Skill that audits any Shopify store for AI visibility and search readiness.

## What it checks

- SEO
- GEO
- AEO
- Technical SEO
- Core Web Vitals
- JSON-LD
- Organization Schema
- Product Schema
- FAQ Schema
- AI Visibility
- Entity Consistency
- Trust Signals
- Internal Linking
- Canonical URLs
- Sitemap
- Robots.txt
- llms.txt
- Conversion Readiness

## Example Prompt

Audit https://www.example.com

Generate:

- Executive Summary
- GEO Score
- SEO Score
- AI Visibility Score
- Priority Fixes

- Claude Desktop
- Claude Code
